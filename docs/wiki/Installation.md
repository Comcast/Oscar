Run the following command: 

    java -jar oscar.jar

This will create the following directories at the location of the JAR file:
* certificates - Folder that contains Manunfactor and Co-Sign CVC
* db - SQLite TLV dictionary (Provided)
* mibs/txt - [MIBS DOCSIS and PacketCable **Download Required**](https://comcast.github.io/Oscar/mibs/mibs.zip)
